## LinkedIn Organic Content Strategy for B2B SaaS — Research Project

> Why This Topic

LinkedIn is currently the highest-signal channel for B2B SaaS companies — especially at the early stage, where paid acquisition is expensive and brand credibility matters. Organic content on LinkedIn can drive pipeline, attract inbound leads, and build founder authority without ad spend.

I chose this topic because it sits at the intersection of content strategy and distribution, and because the gap between "generic LinkedIn tips" and "actual B2B SaaS playbooks" is wide. The goal of this research is to close that gap by going directly to practitioners who have built real audiences and pipelines through LinkedIn.

> Collection Method

- **LinkedIn posts:** Manually collected from each expert's public profile (5–10 most recent posts). LinkedIn restricts API access for post content, so this was done by hand. Each post is saved as a `.md` file with date and engagement notes where visible.
- **YouTube transcripts:** Collected using a Python script with the `youtube-transcript-api` library. No API key required. Script is in `/research/other/get_transcripts.py`.
- **Other materials:** Newsletter issues and blog posts collected via direct URL where publicly available.
