3. Problems & Fixes
I had a bit of trouble figuring out where to log in for the Claude and Codex extensions. I basically clicked through all the menus and checked their official websites until I found the right login button. Problem solved!

I kept getting a "failed to create file" error when trying to clone a repository directly through Cursor. I decided to install the standalone Git app on my system. This apparently "enabled" Cursor's Git integration, and after that, everything worked perfectly and I could manage my files without any more errors.
